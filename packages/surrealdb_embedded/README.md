# surrealdb-embedded

This package provides the **embedded SurrealDB engine** for the [`surrealdb`](https://pypi.org/project/surrealdb/) Python SDK.

Most users should install the main SDK:

```bash
uv add surrealdb
```

To enable embedded URLs (`memory`, `mem://`, `file://`, `surrealkv://`), install:

```bash
uv add "surrealdb[embedded]"
```
